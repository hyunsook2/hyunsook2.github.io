Project

조직 간의 원활한 의사소통 및 효율적인 협업을 위한 다양한 비즈니스 기능을 지원하는 웹 페이지입니다.
교육청에서 쓰던 업무용 소프트웨어를 보고 아이디어를 얻어 개발하였습니다. 팀 프로젝트를 진행하면서 협업 하는 과정 에 관해 많이 배울 수 있었습니다.
요구사항 분석, Use Case 명세서, Use Case Diagram, Ckass Diagram, ERD Diagram, UI 와 같은 필요한 명세를 작성하며 소통하였고, 필요한 지식도 함께 공유하며 학습할 수 있었습니다.
클라이언트의 요청을 처리하고 그 결과를 반환하는 서버의 컴포넌트인 Jaca Servlet 을 통해 동적안 웹페이지를 구성하였습니다.

또한, Model, View, Controller 즉, MVC2 패턴을 적용하여, 웹 애플리케이션의 구조가 명확해지고, 유지보수 및 확장성이 용이해지는 장점을 경험하여 웹 개발에 대한 전반적인 이해도를 높이는 데 큰 도움이 된 프로젝트 경험이었습니다.

주요 기능

출결시스템, 근태현황 조회, 전자결재 시스템, 조직도 조회, 프로필 관리
계정 생성, 계정 대량생성 [양식에 맞는 파일 등록],유저 생성


🔨 Technology Stack(s)

Frontend : JSP, CSS3, JavaScript
Backend : Java, Python
Framework : Spring
Design Pattern / Architecture : Servlet, MVC2 Pattern
Database : Oracle
Version Control : Git, GitHub
